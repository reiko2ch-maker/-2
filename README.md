# 宵宿 - PS1風ローポリ旅館ホラー

GitHub Pagesにそのまま置いて遊べる、軽量な一人称3Dホラー試作版です。

## ファイル構成
- `index.html`
- `styles.css`
- `game.js`
- `README.md`

## 遊び方
### PC
- 移動: `W A S D`
- 視点: 画面ドラッグ
- 調べる / 話す: `E` / `Enter`

### スマホ
- 左下パッド: 移動
- 画面右側スワイプ: 視点操作
- `ACT`: 調べる / 話す

## 内容
- 古い和風旅館を一人称で歩ける
- 昼は接客、夜は見回り
- 低ポリのPS1風表現
- ローカル保存対応
- 写真要素をもとにした、白ヘルメット・青制服・紅白旗のローポリ怪異NPCを実装

## GitHub Pagesに置く手順
1. 新規リポジトリを作る
2. この4ファイルをリポジトリ直下にアップロードする
3. `Settings > Pages` を開く
4. `Deploy from a branch` を選ぶ
5. `main` / `root` を選んで保存
6. 表示されたURLを開く

## 注意
- 画像や音声ファイルを使わず、コードだけで軽量に作ってあります
- スマホブラウザ向けに軽くしているので、超高精細ではなくPS1風の粗い見た目に寄せています
- さらに強化したい場合は、部屋数追加、接客タスク追加、エンディング追加、モデル差し替えで拡張できます
