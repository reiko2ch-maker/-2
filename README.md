# 宵宿 v13

PS1風の荒い質感を残しつつ、立体感とエリア移動を強化した一人称ホラー試作です。

## ファイル
- `index.html` タイトル画面
- `play.html` ゲーム本体
- `styles.css` UIと画面スタイル
- `game.js` ロジックと描画
- `README.md` この説明

## 今回の改善
- 帳場 / 客室廊下 / 201号室 / 北の裏廊下 のエリア移動
- 昼の仕事をしながら客から不可解な話を聞く導線
- 夜に事件調査へ移る流れ
- 旅館らしい扉・障子・帳場・提灯の見た目強化
- 目標とミニマップで進行を分かりやすく改善

## 操作
- スマホ: 左スティック移動 / 右側ドラッグで視点 / ACTで会話・調査・移動
- PC: WASD / 右側ドラッグ視点 / E or Space で会話・調査
