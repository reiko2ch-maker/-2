## 遊び方
### PC
- 移動: `W A S D`
- 視点: 画面ドラッグ
- 調べる / 話す: `E` / `Enter`

### スマホ
- 左下パッド: 移動
- 画面右側スワイプ: 視点操作
- `ACT`: 調べる / 話す

## GitHub Pagesに置く手順
1. 今の4ファイルを削除または上書き
2. この4ファイルをリポジトリ直下にアップロード
3. `Settings > Pages`
4. `Deploy from a branch`
5. `main` / `root`
6. 公開URLを再読み込み

## 注意
- 画像や音声ファイルを使わず、コードだけで軽量に作っています
- PS1風の粗い見た目を狙った軽量版です
