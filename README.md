# 宵宿 v16 Overdrive Web

GitHub / itch.io の軽量Webテスト向けに、容量を気にせずできるだけ盛った長編寄り版です。

更新点
- 横画面前提のまま、中央の「横画面推奨」表示を削除
- iPhone Safari 向けに、ピンチ・ダブルタップ拡大を抑える処理を追加
- DAY2 を追加
- 従業員室 / 203号室 / 中庭 / 離れ通路 を追加
- 宿帳庫の追跡に加えて、離れ通路の第二追跡を追加
- 会話中の人物ポートレートを少し大きく、顔が分かりやすい形に調整
- MENU から SAVE / LOAD / HUD ON-OFF / TITLE

ファイル
- index.html
- play.html
- styles.css
- game.js
- README.md

公開方法
1. 既存ファイルを削除
2. この5ファイルだけをリポジトリ直下へアップロード
3. GitHub Pages か itch.io の HTML5 で確認

注意
- こちらでは iPhone 実機そのものは確認できません
- 今回は「Webで触れる範囲の上限寄り」を狙った版です
- さらに本格的な超高品質版は、itch.io のダウンロード版や Unity / Unreal 向けが本命です
