# 宵宿 v15 GitHub Edition

GitHub Pages でそのまま動かせる横画面ホラー試作の拡張版です。

## 追加したもの
- ストーリーを拡張
- 202号室 / 厨房 / 閉鎖棟を追加
- 深夜パートで「誘導員」の追跡イベントを追加
- キャラクターを顔つきの分かりやすい高密度ビルボードに変更
- 会話時のポートレート表示を追加

## 使い方
リポジトリ直下に以下の5ファイルを置いて GitHub Pages を公開してください。
- index.html
- play.html
- styles.css
- game.js
- README.md

## 操作
- PC: WASD / マウス / E / ESC
- スマホ: 左スティック / 右側ドラッグ / ACT / MENU
