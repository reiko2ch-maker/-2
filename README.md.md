# 宵鳴り旅館

スマホ縦画面向けの、和風旅館×住み込み勤務×接客シミュ×探索ホラーのシングルページWebゲームです。

## 特徴
- 昼は接客・清掃・帳場整理などの選択式シミュレーション
- 夜は一人称風の探索パート
- 選択肢と行動で分岐するマルチエンディング
- GitHub Pages にそのまま置ける静的ファイル構成
- localStorage セーブ対応
- 「青い制服 / 白ヘルメット / 紅白の旗」の誘導員怪異キャラを実装

## ファイル構成
- `index.html`
- `styles.css`
- `game.js`

## GitHub Pages で遊ぶ手順
1. 新しい GitHub リポジトリを作成
2. この3ファイルをそのままアップロード
3. `Settings` → `Pages` → `Deploy from a branch`
4. Branch を `main` / root に設定
5. 数十秒後に発行されるURLへアクセス

## ローカルで確認する方法
- `index.html` をブラウザで直接開いても動作します
- iPhone / Android でもブラウザで起動可能です

## 追加で伸ばしやすい点
- 画像素材を入れて3D寄りにする
- BGM / SE を追加する
- 客の会話イベントを増やす
- 実績 / エンディング回収要素を追加する
- 実写風の立ち絵や低ポリ3Dモデルに差し替える
